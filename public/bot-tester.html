Comment utiliser — Multi-Groupes

  1. Cliquez + Ajouter un groupe pour creer un groupe de robots
  2. Entrez le code du match (affiche sur la page professeur, ex: 6ADEUP)
  3. Choisissez le nombre de robots et la vitesse
  4. Cliquez Lancer sur le groupe — les robots rejoignent le match
  5. Repetez pour chaque match/phase que vous voulez tester

Les 4 phases du tournoi

  • Phase 1 CRAZY WINNER CLASSE — Matchs intra-classe
  • Phase 2 CRAZY WINNER ECOLE — Gagnants de chaque classe
  • Phase 3 CRAZY WINNER CIRCONSCRIPTION — Gagnants de chaque ecole
  • Phase 4 CRAZY WINNER ACADEMIQUE — Finale regionale
