Comment utiliser

  1. Ouvrez le jeu normalement dans un autre onglet
  2. Mode Multijoueur : Creez une salle, notez le code, et entrez-le ici
  3. Mode Arena/Training : Lancez un match depuis le dashboard professeur, notez le Match ID
  4. Choisissez le nombre de robots et leur vitesse
  5. Cliquez Lancer les robots — ils rejoignent et jouent automatiquement
  6. Jouez contre eux dans votre onglet de jeu !
